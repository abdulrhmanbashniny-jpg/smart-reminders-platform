# منصة التذكيرات الذكية (Smart Reminders Platform)
**الجهة:** مصنع جدة للدهانات والمعاجين (Jeddah Paint Factory)  
**الهدف:** تسهيل عمل الموظفين عبر تذكيرهم بالمواعيد والانتهاءات والالتزامات، ودعمهم بإجراءات واضحة (استلام → تنفيذ → اعتماد → إغلاق)، بما يرفع الإنتاجية ويقلل نسيان المهام تحت ضغط العمل.

---

## لماذا هذه المنصة؟
في بيئة العمل السريعة، كثير من المهام تتأخر ليس بسبب الإهمال بل بسبب كثرة الانشغال. المنصة تضمن:
- تذكيرات ذكية قبل الموعد (Due/Expiry)  
- استلام إلزامي للمهام (Acknowledgement)  
- تصعيد منظم عند عدم الاستلام (مشرف → مدير القسم)  
- سجل تدقيق كامل لكل تعديل/اعتماد  
- قنوات إشعار: داخل النظام (إلزامي) + WhatsApp + Telegram (اختياري)

---

## الأدوار والصلاحيات (مختصر)
- **مدير النظام (System Admin):** صلاحيات كاملة + إعدادات النظام + التقارير والتحكم
- **المدير العام (General Manager):** صلاحيات عالية جدًا شبه مدير النظام
- **مدير القسم (Department Manager):** إدارة ومتابعة قسمه فقط
- **المشرف (Supervisor):** متابعة فريقه واعتماد الإغلاقات افتراضيًا
- **الموظف (Employee):** مهامه فقط + استلام/تنفيذ/طلبات

---

## رحلة العمل (Workflow)
1) **إنشاء عنصر** (Item)  
2) **تم الاستلام** إلزامي  
3) **التذكير والتصعيد**:
   - Day 0: إشعار للمشرف إذا لم يتم الاستلام
   - +24 ساعة: تصعيد لمدير القسم
4) **طلب إغلاق** → اعتماد (المشرف افتراضيًا، وعند التصعيد يعتمد المدير)
5) **إغلاق** مع سبب إلزامي + توثيق في سجل التدقيق
